## Installation

 ```bash
   # Linux / macOS
   chmod +x install.sh
   ./install.sh
 ```

 ```powershell
   # Windows (PowerShell)
   .\install.ps1
   # If symlinks fail, enable Developer Mode or run as Admin
 ```

```bash
git clone https://github.com/TheLinuxers/dotfiles.git ~/.config
cd ~/.config
```
